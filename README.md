# DevOps Monitor - API de Monitoramento de Serviços

## 🚀 Acesse online
https://projeto-devops-okd5.onrender.com

## 📊 Endpoints
- `/health` - Status da API
- `/monitor` - Monitora GitHub e Google

## 🛠️ Tecnologias
- Node.js + Express
- Docker
- GitHub Actions (CI/CD)
- Render.com (Deploy)
